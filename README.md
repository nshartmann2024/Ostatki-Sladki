# Остатки Сладки

Подбор рецептов по ингредиенту (венская домашняя кухня). Web (Streamlit) + Telegram-бот. Поиск по форме, SEO-страницы по `?ingredient=...`, sitemap по `?sitemap=1`. Подробный статус: **PROJECT_STATUS_FEB_2026.md**.

## Запуск Web (Streamlit)

Из корня проекта:

```bash
streamlit run adapters/web/app.py
```
