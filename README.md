# Остатки сладки — что приготовить из того, что осталось

Сервис, который помогает быстро найти, что приготовить из хлеба, картофеля, яиц и других продуктов, которые уже есть дома.

👉 Попробовать онлайн (бесплатно, без регистрации):
https://ostatki-sladki.streamlit.app👉

---

## 💡 Идея проекта

У вас остался хлеб, картофель, яйца или сыр — и непонятно, что из этого приготовить?

Введите 1–2 ингредиента — и получите простые рецепты, которые можно приготовить прямо сейчас.

Проект вдохновлён венской (австрийской) домашней кухней, где из остатков продуктов создают полноценные блюда — без лишних затрат и сложных ингредиентов. Подходит для быстрых домашних рецептов и повседневной кухни.

---

## 🔍 Что умеет сервис

- Поиск рецептов по ингредиенту
- Простые блюда из остатков
- Что приготовить из хлеба, картофеля, яиц и других продуктов
- Быстрые рецепты без лишних шагов
- Доступен как сайт на Streamlit и как Telegram-бот

## 🧱 Архитектура

Проект построен по принципу:

- `core/` — вся бизнес-логика (поиск, рецепты)
- `adapters/telegram` — Telegram-бот
- `adapters/web` — Streamlit сайт

Один core → несколько интерфейсов.
